Configuração inicial
Conecte sua conta Google e a IA da Anthropic. Suas credenciais ficam salvas só no seu navegador.
Passo 1 — Google OAuth
Client ID do Google Cloud
console.cloud.google.com → APIs & Services → Credentials → OAuth 2.0 Client ID (Web). Em "Authorized JS origins" coloque a URL deste site no Netlify.
Passo 2 — Anthropic
Chave da API Anthropic
console.anthropic.com → API Keys → Create Key. Começa com sk-ant-
Suas chaves são salvas no localStorage e nunca enviadas a terceiros além das APIs oficiais.
